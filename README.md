# ShoppingLists API

Серверный компонент системы управления списками покупок с поддержкой совместной работы в рабочих пространствах.

## Основные возможности

- Аутентификация пользователей (включая JWT-токены и refresh-токены)
- Создание и управление персональными и совместными списками покупок
- Работа в рамках рабочих пространств с приглашениями и управлением членами
- Синхронизация изменений между участниками рабочего пространства

## Технологический стек

- **FastAPI** — веб-фреймворк и API
- **SQLAlchemy** — ORM для работы с базой данных
- **PostgreSQL** — реляционная база данных
- **Alembic** — миграции базы данных
- **Pydantic** — валидация данных и схемы
- **Uvicorn** — ASGI-сервер
- **bcrypt** — хеширование паролей
- **PyJWT** — работа с JWT-токенами
- **Docker** — контейнеризация приложения
- **Docker Compose** — оркестрация (база данных, pgAdmin, приложение)

## Запуск проекта

### Локальный запуск

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Настройте переменные окружения через файл `.env` (используйте `.env.example` как образец).

3. Примените миграции базы данных:
   ```bash
   alembic upgrade head
   ```

4. Запустите приложение:
   ```bash
   python src/main.py
   ```

## Структура проекта

- `src/` — основной код приложения
  - `api/` — маршруты 
  - `core/` — базовые конфигурации, логирование, безопасность
  - `database/` — модели, репозитории, сессии и настройки БД
  - `schemas/` — Pydantic-схемы для валидации
  - `services/` — бизнес-логика приложения
  - `utils/` — вспомогательные утилиты
- `tests/` — модульные и интеграционные тесты
- `requirements.txt` — зависимости
